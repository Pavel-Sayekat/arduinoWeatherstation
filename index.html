    • Arduinoweatherstation

    weather station based on arduino. Developed on Uno and Mega.

    Project maintained by chiemseesurfer Hosted on GitHub Pages — Theme by mattgraham

    This is an arduino weatherstation without its own webserver. I use a Arduino Uno. The Arduino Uno is a microcontroller board based on the ATmega328.

    Supported sensors:

    • Temperature

    • Pressure

    • Humidity

    • Windspeed

    • Winddirection
    • rainfall
